# Kinematic Projectile Vectoring in a Metroid-Style Game

## Overview

This project is a 2D Metroid-style game developed in Godot that focuses on implementing a kinematic projectile vectoring system. The project explores directional shooting mechanics, projectile physics, collision detection, and responsive player controls commonly found in action-platformer and Metroidvania games.

The primary objective was to design a flexible projectile system capable of firing in multiple directions while maintaining smooth gameplay and accurate collision handling.

---

## Features

* Directional projectile shooting
* Kinematic projectile movement
* Collision detection and response
* Player movement and combat mechanics
* 2D Metroid-style gameplay
* Sprite-based animations
* Modular Godot scene architecture

## Technical Highlights

### Projectile Vectoring System

The projectile system uses vector mathematics to calculate movement direction and velocity. This enables projectiles to travel accurately in multiple directions while maintaining consistent speed and collision behavior.

### Collision Handling

Projectiles interact with the environment and game entities through Godot's physics and collision systems, ensuring reliable detection and response.

### Modular Design

The project is structured using reusable scenes and scripts, making it easy to extend projectile types, weapons, and combat mechanics.
